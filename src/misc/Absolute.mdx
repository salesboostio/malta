# Absolute

정상적인 document flow를 벗어난 곳에 컴포넌트를 위치시키고 싶을 때 사용하는 wrapper.  `<Relative>` 블락 안에서만 적용된다. 
`<Absolute>` 내부에 있는 컴포넌트의 width, height는 그 부모 컴포넌트의 전체 width, height 계산에서 제외되며
당연히 padding, margin 속성은 적용되지 않는다.

top, left, right, bottom의 offset은 부모 컴포넌트를 기준으로 계산된다.
