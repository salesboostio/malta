# HyperLink

HyperLink는 앵커(a)태그로 표현되며, 링크를 표현하는 인라인 컴포넌트이다. 기본적으로 새 창에 열리게되어있다.
이는 React Router가 Link를 사용하며 새 창이 아닌 기존 윈도우에서 전환되기 떄문이다.
