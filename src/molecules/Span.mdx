# Span

Span은 텍스트 노드 중간에 삽입될 수 있는 인라인 컴포넌트이다. 속성은 Text를 상속받는다

## Dependencies
<App>
<Flex>
<Chip bg="blue.deep" color="#FFF">Text</Chip>
</Flex>
</App>

## Properties

|Property|Type|Default|
|-|-|-|
|italic| boolean | false |
|caps| boolean | false |
|underline| boolean | false |
|through| boolean | false |
|medium| boolean | false |
|regular| boolean | true |
|demiLight| boolean | false |
|light| boolean | false |
|fontSize| ResponsiveValue | h4 |
|lineHeight| ResponsiveValue | 1.4em |
|color| ResponsiveValue | black.standard |
